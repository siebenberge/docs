To interact with zahls.ch you can use our API or the following SDKs. If you develop your own SDK for zahls.ch, let us know and we'll list it here. As we work closely with Payrexx you can reuse the same logik and change the base-url.

<Info>Base URL: https://api.zahls.ch/</Info>

There are currently the following SDKs available:
<CardGroup cols={2}>
  <Card title="NodeJS SDK" href="https://www.npmjs.com/package/zahls-sdk">
    Use our zahls.ch SDK for NodeJS with TypeScript support.
  </Card>
  <Card title="PHP SDK">
    Use our PHP SDK.
  </Card>
</CardGroup>

To get started with our API, and you can't use one of our SDKs please check Authentification first.
